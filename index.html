Working but still WIP

  • JS BigInt and 64 bit WASM Integer integration, no Webkit
  • SIMD, not enabled by default
  • Threads and atomic operations, no Webkit
  • WASI, still experimental

Source: webassembly.org/roadmap

Main features

  • Fast
  • Safe
  • Compute only

WASM targets

  • Browsers: 93% users, WebAssembly MDN docs
  • JS Runtime: Node.js, Deno
  • WASM Runtime: wasmtime, wasmer

Programming Languages

  • C/C++: Emscripten
  • Rust: Working Group
  • AssemblyScript
  • C#: Blazor
  • Other

Why WebAssembly?

  • Applications porting to the Web, e.g. Pyodide
  • Reuse code in JavaScript
  • One runtime for many languages
